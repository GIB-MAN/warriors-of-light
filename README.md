# Warriors of Light

A front-end project built using responsive design principles and a modular SCSS architecture.

---

## 🚀 Project Updates / Improvements

The following updates and improvements were implemented in the project:

1. All icons were re-downloaded with **transparent backgrounds**, organized on the Desktop, and their links were updated.

2. A **hover background effect** was added to the **left sidebar**, working across all **breakpoints**.

3. **Header button sizes** were corrected — they now properly adjust their size and style across all breakpoints.

4. The **right sidebar (Brands Activity)** section was fixed — it smoothly resizes and fully **appears / disappears** when required.

5. The entire codebase was migrated to **SCSS**:
   - **Variables** were created
   - **Mixins** were implemented
   - Mixins were imported into all necessary section files.

6. The code structure heavily utilizes:
   - **SCSS nesting**
   - **Mixins**
   - **Dynamic mixins with variables**

7. A **GitHub repository** was created for the project, and the current version of the website was pushed to the **main branch**.

8. Collaborators were invited to the repository:
   - Nazi Foladashvili  
   - Nukri Otarashvili

9. **Navigation and header menus** were fixed, along with the **Friends Activity show/hide functionality**.

10. **Navigation menu boundaries** were corrected for responsive design, and the **Friends Activity content in the right sidebar** was updated.

11. **Music cards** were added to the content section.

12. A **vertical scrolling effect** was added to the cards section. The **footer** was also fixed.

13. **Hover effects** were added to the cards.

14. The **mobile version template structure** was added.

15. **Animation** was added to the navigation submenu.

16. Additional **hover effects for cards and buttons** were implemented when the mouse cursor is placed over them.

17. A **full backup version of the project** was restored.
